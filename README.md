
# 🌈 Hoya Team Tokyo Pride Trip 2026 🦄

這是一個專為 Hoya 團隊設計的東京旅遊同步手冊！

## ✨ 主要功能
- **多人同步清單**：大家可以一起勾選要帶的行李。
- **即時分帳錢包**：輸入日幣，自動換算台幣並記錄誰欠誰錢。
- **AI 彩虹御神籤**：每天早上抽一張 AI 巫女為你準備的運勢籤。
- **互動行程表**：點擊地點直接開啟 Google Maps 導覽。
- **床位分配器**：隨機分配河口湖膠囊旅館的床位。

## 🛠️ 開發資訊
- **前端框架**: React + Vite + Tailwind CSS
- **資料庫**: Firebase Firestore (即時同步)
- **AI 支援**: Google Gemini API

## 🔗 如何使用
直接打開部署後的網址即可使用！建議將網址「加入主畫面」當作 App 使用。
